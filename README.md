# bitmap-font-putter-cpp

文字コードから、Shift_JIS の順で並んだフォント画像データの文字位置を計算し、  
画像を切り抜いて、指定の文字として画面に表示します。  

![Screenshot](https://github.com/wertrain/bitmap-font-putter-cpp/blob/master/Screenshots/00.png)

### 関連
このプロジェクトで使用する画像データは以下のツールで出力しました。
